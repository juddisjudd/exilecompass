# Tauri + SvelteKit + TypeScript

This template should help get you started developing with Tauri, SvelteKit and TypeScript in Vite.

## Build Commands (from workspace root)

- `bun run build:frontend` builds the Svelte frontend into `build/`
- `bun run build:backend` builds the Rust/Tauri backend in release mode
- `bun run build:release` runs both frontend + backend release builds
- `bun run release` runs `tauri build` to produce distributable app bundles
